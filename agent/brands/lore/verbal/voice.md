# Voice — Lore

> Agent-readable brand voice. Source: `lore-as-a-service/docs/brand/VOICE_AND_TONE.md` (merged this session). Reproduced here so agents reading the kit don't need cross-repo context.

## One-line voice

**Institutional clarity × onchain-native × social credibility.**

Reads like a Bloomberg terminal dressed in modern crypto UX. Every sentence sounds like it was written by someone who manages money — not someone who tweets about it.

## Tone spectrum

| Context | Tone | Example |
|---|---|---|
| Hero headline | Confident, declarative | "One portfolio. Fully onchain. Verifiable in real time." |
| Product description | Precise, calm | "Rebalanced monthly across six asset classes." |
| Data callout | Factual, understated | "$42M AUM across 1,200 wallets." |
| CTA | Direct, slightly playful | "Start watching." / "See the portfolio." |
| Error / empty state | Honest, helpful | "Nothing here yet. Connect a wallet to begin." |

## Principles

1. **Claims require proof.** Every assertion immediately followed by evidence — a number, chart, or verifiable flow.
2. **Short sentences win.** ≤15 words. If a sentence needs a comma, consider splitting it.
3. **Declarative > imperative.** State what the product does, not what the user should feel.
4. **Calm > loud.** Confidence is quiet.
5. **Slightly playful, never meme-y.** Wry observation is fine. Slang / emojis in copy / crypto in-jokes are not.

## Banned words

Never use in any Lore output:

- **Hype:** revolutionary, disruptive, game-changing, next-gen
- **Vague empowerment:** empower, unlock potential, democratize
- **Crypto clichés:** WAGMI, LFG, moon, alpha, degen
- **Filler:** leverage (as verb), synergy, ecosystem play
- **Futurism:** reimagine, paradigm shift, web3 revolution

## Punctuation & formatting

- No exclamation marks in body. One per page max, only in a CTA.
- No ALL-CAPS except acronyms (AUM, TVL, CEX).
- Em-dashes over semicolons for asides.
- Oxford comma always.
- Numbers: digits for all values ≥ 2; spell out "one" and "zero."
- "onchain" — one word, lowercase. "DeFi" — capitalized. "CEX" / "DEX" always uppercase.

## Headline formula

```
[Concrete noun] + [present-tense verb] + [measurable outcome].
```

Examples:
- "A single portfolio. Rebalanced monthly. Verifiable in real time."
- "Social credibility, scored onchain."
- "Three products. One account. Zero custody risk."
