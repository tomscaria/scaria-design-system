# earth-dark

Inversion of `earth-light`. Same brand identity, dark graphite surface, parchment ink. Lime accent reads as the hero element rather than a supporting one (because there's no white space competing with it).

## When to use

- Product dashboards (default)
- Operator-facing tools
- Anywhere a "terminal" feel is appropriate
- Late-stage deck slides for emphasis (paired with `data-expression="marketing"`)

## Pairs with

`earth-light` — same brand, opposite mode. They share `--accent` (lime), `--fg-inverted` ↔ `--fg`, and type scale + spacing defaults.

## Inversion rules

- `--bg` ↔ `--fg-inverted`
- `--fg` ↔ `--bg-inverted`
- `--brand` flips: graphite on light becomes lime on dark
- Shadows are honest blacks, not warm browns
- `--accent` value stays identical across both modes
