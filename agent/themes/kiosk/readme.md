# kiosk

Brutalist newsstand. Sharp corners, electric yellow accent, hard-edge shadows, Syne display. Anti-design as design. Restored from earlier expressive themes work — was lost when the expressive themes file was skipped in the unification PR; reauthored here in atomic-kit shape.

## When to use

- Pricing pages
- One-pagers, manifestos, statements of intent
- "Here's what we believe" deck slides
- Anywhere the message benefits from being shouted, not whispered

## Avoid for

- Dashboards, data-dense product UI
- Anywhere users dwell — the visual intensity doesn't reward dwell time
- Documentation (use `primitive` instead)

## Distinguishing features

- **No radii.** `--r-* : 0`. Everything is square.
- **Hard-offset shadows.** `4px 4px 0 black`. No blur.
- **Syne display.** Sharp display sans. No serifs.
- **Yellow accent.** `hsl(63 100% 58%)` — electric, not soft.
- **Terse motion.** 200ms entries, linear easing. No springs.

## Pairs with

Itself. Kiosk has no dark inversion — the harsh light is the point.
