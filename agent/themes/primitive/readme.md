# primitive (aka primitive-light) — THE STUDIO

**v2 — repositioned.** Primitive is no longer the brand-neutral Inter baseline; it is the **Scaria flagship marketing theme** and the default public identity (tomscaria.com). It merges the best of the house brands into one register:

- **From Lore:** the material — parchment `#F0ECE1`, chartreuse `#D4F24B`, Lock Serif display, Aeonik/Aeonik Mono.
- **From Revenant:** the discipline — ink-filled actions, one accent per view, tight radii (2–12px), mono letterspaced kickers.
- **Its own:** the place — the painted trading studio (hero plate), terminal amber `#E8A33D` for market/live data, moss glow `#A9BD85`, paper grain.

Mood: *Bloomberg terminal, but clean.* Editorial calm carrying operator density.

## When to use

- The public site (default theme) and any marketing/portfolio surface
- Case studies, memo hubs, share links that should read "Scaria" without naming a venture brand
- Anywhere the old advice said "use earth-light for product" but the surface is *personal-brand* first

## Rules

- Chartreuse gets ONE hero moment per view (wordmark chip, dial ring, or CTA — not all three).
- Terminal amber is **data garnish only**: tickers, live numerals, market glyphs. Never headings, never buttons.
- Display serif for names and claims; mono caps for wayfinding; sans for everything else.
- The brand layer (cursor, grain, dial, kicker, ticker — `agent/visual/patterns/brand-layer.css`) stays mounted across ALL themes; only tokens re-light it.

## Baseline duty (retained)

Primitive still serves as the component baseline for docs/screenshots — it is simply no longer *neutral*. If you need a truly brand-free surface, screenshot with tokens overridden inline.

## Pairs with

`primitive-dark` (Studio Night).
