Methodology

What makes an agent survive.

The apparatus isn't trained on historical regimes. It updates its prior as new evidence arrives. Every agent carries a belief state — not a fixed parameter set.

P.01

Prior Updating
over prediction

Agents don't predict future prices. They maintain and update a prior over market regimes. Uncertainty is a first-class input, not something to minimize away.

P.02

Survival as the
fitness function

No optimization target except continued operation. An agent that loses capital fails the only test that matters. No backtested Sharpe — field performance only.

P.03

Population
over individual

No single agent carries the fund. The apparatus distributes across the population. Catastrophic loss in one agent is bounded and expected. That's by design.

P.04

All-weather
design

No regime assumptions. Agents that thrive only in trending markets fail Canary when conditions shift. The graveyard is full of momentum strategies. The survivors aren't.

Every agent that dies feeds the ones that don't. The apparatus doesn't mourn failure — it indexes it.
